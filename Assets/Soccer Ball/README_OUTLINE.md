# サッカーボール アウトライン実装

## 📝 概要

サッカーボールに黄色の輪郭線（アウトライン）を追加して視認性を向上させました。

## 🎨 実装内容

### 1. シェーダー

- **ファイル**: `Assets/Shaders/OutlineShader.shader`
- **機能**: URP 対応の 2 パスアウトラインシェーダー
  - Pass 1: アウトライン描画（法線方向に頂点を押し出し）
  - Pass 2: 通常のライティング描画

### 2. マテリアル

新しく 2 つのマテリアルを作成：

#### White Material (白い部分用)

- **ファイル**: `Assets/Soccer Ball/Materials/SoccerBall_Outline_White.mat`
- **Base Color**: 白 (1, 1, 1)
- **Outline Color**: 黄色 (1, 1, 0)
- **Outline Width**: 0.03

#### Black Material (黒い部分用)

- **ファイル**: `Assets/Soccer Ball/Materials/SoccerBall_Outline_Black.mat`
- **Base Color**: ダークグレー (0.1, 0.1, 0.1)
- **Outline Color**: 黄色 (1, 1, 0)
- **Outline Width**: 0.03

### 3. プレハブ更新

`Assets/Soccer Ball/Prefabs/Soccer Ball.prefab`のマテリアルを新しいものに差し替えました。

## ⚙️ カスタマイズ方法

### アウトラインの色を変更

マテリアルの Inspector で `Outline Color` を調整：

- 黄色: (1, 1, 0)
- オレンジ: (1, 0.5, 0)
- 赤: (1, 0, 0)
- 緑: (0, 1, 0)

### アウトラインの太さを変更

`Outline Width` パラメータで調整（推奨範囲: 0.01 ～ 0.05）：

- 細い: 0.01 ～ 0.02
- 標準: 0.03（デフォルト）
- 太い: 0.04 ～ 0.05

### ボール本体の色を変更

`Color` パラメータで調整可能

## 🎮 使用方法

1. Stadium シーンにサッカーボールを配置
2. プレハブから `Soccer Ball` を選択
3. 自動的にアウトラインが表示されます

## 🔧 トラブルシューティング

### アウトラインが表示されない場合

1. URP が正しく設定されているか確認
2. マテリアルのシェーダーが `Custom/URP Outline` になっているか確認
3. Outline Width が 0 になっていないか確認

### パフォーマンスが気になる場合

- Outline Width を小さくする（0.01 ～ 0.02）
- より遠い距離ではアウトラインを無効化する LOD システムの導入を検討

## 📊 パフォーマンス

- **追加描画パス**: 1 パス（アウトライン用）
- **追加ポリゴン**: なし（既存メッシュを利用）
- **WebGL 対応**: ✅ はい
- **モバイル対応**: ✅ はい
